# Convert tfrmt elements

This collection of functions is used to create a json or json ready
objects. While primarily intended for internal use these functions can
be used externally

## Usage

``` r
as_json(x)
```

## Arguments

- x:

  tfrmt or tfrmt element

## Value

as_json.tfrmt() will return a json object all other methods will return
a list that is cleaned so it can be directly converted to a json object
